This page points to the Eclipse update site for Groovy-Eclipse.

To use this page as an update site:

  1. Copy the URL for the site (without the index.html at the end of it).
  2. In your Eclipse installation, go to Help -> Install new software....
  3. In the work with box, paste the URL you just copied.
  4. Select all of the components you want to install.
  5. Click next and follow the instructions.
  6. Wait for the install to complete and restart.

Downloading the archived Groovy-Eclipse update site

Click below to download an archived version of the update site:

@GROOVY_ECLIPSE_VERSION@

To use the archived update site:

  1. Download the zip file from the link above.
  2. In your Eclipse installation, go to Help -> Install new software....
  3. In the work with box, click Add and then Archive
  4. Choose the zip file you just downloaded.
  5. Select all of the components you want to install.
  6. Click next and follow the instructions.
  7. Wait for the install to complete and restart.

See the Groovy-Eclipse wiki for information on how to install.
